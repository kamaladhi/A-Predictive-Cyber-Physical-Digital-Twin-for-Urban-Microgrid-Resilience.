Key Architecture Features
Clean Separation of Concerns:

EMS receives MicrogridMeasurements from components
EMS returns EMSControlOutputs with commands only
Zero physics simulation or component modification

Operation Modes:

GRID_CONNECTED
ISLANDED
TRANSITION_TO_ISLAND
TRANSITION_TO_GRID
FAULT

Core Control Functions:

Grid Fault Detection - Voltage/frequency monitoring with configurable thresholds
Islanding Logic - Fast detection (<100ms logic) with safe transition
Battery Dispatch - Mode-specific strategies (grid-connected vs islanded)
Generator Dispatch - Two-tier control (Gen1=critical, Gen2=non-critical) with min on/off times
Load Shedding - Priority-based shedding that NEVER touches critical loads (320 kW)
Reconnection Logic - Grid stability verification with configurable delay

Safety-First Design:

Critical loads (320 kW) are hardcoded as unshedable
Generator minimum on/off times enforced
Battery SoC thresholds for generator auto-start/stop
Aggressive load shedding at low SoC (<15%)
Time-based restoration delays to prevent oscillation
